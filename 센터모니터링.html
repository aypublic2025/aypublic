🚀 Google Apps Script 연동 설정 방법

아래 절차를 따라 무료 Google 계정으로 센터 모니터링 서버를 5분 만에 구축할 수 있습니다.

1
Google Sheets 만들기
sheets.google.com 에서 새 스프레드시트를 만들고, 주소창의 URL에서 /d/ 와 /edit 사이의 긴 문자열을 복사해두세요.
예: https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms/edit
2
Apps Script 열기
스프레드시트에서 상단 메뉴 확장 프로그램 → Apps Script를 클릭하세요.
3
아래 코드를 붙여넣기
Code.gs의 기존 내용을 전부 지우고 아래 코드를 붙여넣은 후, YOUR_SPREADSHEET_ID 부분을 1단계에서 복사한 ID로 교체하세요. 
// ─── Google Apps Script 코드 ───────────────────────────────────
const SHEET_ID = 'YOUR_SPREADSHEET_ID'; // ← 여기에 스프레드시트 ID 입력

function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    const ss = SpreadsheetApp.openById(SHEET_ID);
    const sheetName = (data.orgCode || 'UNKNOWN').toString().substring(0, 30);
    let sheet = ss.getSheetByName(sheetName);
    if (!sheet) sheet = ss.insertSheet(sheetName);
    sheet.clearContents();
    sheet.getRange(1, 1).setValue(e.postData.contents);
    sheet.getRange(1, 2).setValue(new Date().toISOString());
    sheet.getRange(1, 3).setValue(data.orgName || '');
    return ContentService
      .createTextOutput(JSON.stringify({ success: true, orgCode: data.orgCode }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({ success: false, error: err.message }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  try {
    const ss = SpreadsheetApp.openById(SHEET_ID);
    const result = {};
    ss.getSheets().forEach(sheet => {
      const val = sheet.getRange(1, 1).getValue();
      const syncedAt = sheet.getRange(1, 2).getValue();
      if (val) {
        try {
          result[sheet.getName()] = {
            data: JSON.parse(val),
            syncedAt: syncedAt
          };
        } catch(err) {}
      }
    });
    const jsonStr = JSON.stringify(result);
    // JSONP 지원: ?callback=함수명 파라미터가 있으면 함수로 감싸서 반환
    const cb = e.parameter && e.parameter.callback;
    if (cb) {
      return ContentService
        .createTextOutput(cb + '(' + jsonStr + ')')
        .setMimeType(ContentService.MimeType.JAVASCRIPT);
    }
    return ContentService
      .createTextOutput(jsonStr)
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({ error: err.message }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
4
배포하기
① 저장(Ctrl+S) → ② 오른쪽 상단 배포 → 새 배포 클릭
③ 유형: 웹 앱 선택 → 실행 대상: 나 → 액세스 권한: 모든 사용자
④ 배포 클릭 → 권한 부여 후 웹 앱 URL을 복사
5
URL 배포 및 등록
① 복사한 URL을 이 페이지 대시보드 탭의 URL 입력창에 붙여넣기
② 같은 URL을 각 단체의 지출결의서.html → 기본정보 설정 → 센터 연동 URL에도 입력하도록 안내
③ 이후 각 단체가 결의서를 저장할 때마다 자동으로 데이터가 전송됩니다
💡 오프라인(JSON 파일) 방식: Apps Script 설정 없이도 각 단체가 📤 JSON 내보내기 버튼으로 파일을 이메일/카카오톡 등으로 보내면, 이 페이지의 JSON 불러오기 탭에서 파일을 불러와 현황을 확인할 수 있습니다.
